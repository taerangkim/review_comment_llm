
# AI 기반 리뷰 자동 답글 생성 및 추천 서비스
 
온라인 매장 및 플랫폼 운영자를 위한 AI 리뷰 답글 자동 생성·추천 서비스입니다.
고객 리뷰를 KoBERT + LLM으로 분석하고, RAG 기반 응답 생성으로
사장님이 선택·수정 가능한 자연스러운 답글을 제공합니다.

## 프로젝트 개요

일일이 리뷰에 대한 댓글을 작성하는 것은 시간과 노력이 많이 소요됩니다.
본 프로젝트는 리뷰 내용을 자동으로 분석하여 여러 개의 답글 후보를 추천하고,
운영자가 원하는 답글을 선택·수정하여 빠르게 등록할 수 있도록 돕는 서비스입니다.

## 목표
```
1. 리뷰 답글 작성 시간 대폭 절감
2. 긍정/부정 리뷰에 맞는 일관된 응대 품질 유지
3. 사장님 말투 및 브랜드 톤 반영
4. AI 자동화 + 사람 선택 구조로 신뢰성 확보
```
## 개발 환경
### Fronted 

**React**

    - 리뷰 목록 및 답글 작성 UI
    - 답글 추천 선택 / 수정 인터페이스
    - 감정·키워드 시각화

### Backend
**Flask**

    - 리뷰 데이터 API
    - AI 분석 및 RAG 응답 파이프라인 제어 

### AI / ML
**KoBERT**

    - LLM
    - LangGraph
    - RAG (Retrieval-Augmented Generation)

## 시스템 아키텍처

```
리뷰 입력
   ↓
KoBERT + LLM 감정 / 의도 분석
   ↓
LangGraph (분석 → 검색 → 흐름 제어)
   ↓
RAG 기반 응답 생성
   ↓
다수의 답글 후보 추천
   ↓
사장님 선택 / 수정 / 등록
```

## 핵심 기능

## 1️⃣ 답글 말투 설정 (Reply Tone Configuration)

운영자는 답글 설정 화면에서 원하는 **답글 말투(Tone)** 를 선택할 수 있으며,  
선택된 말투는 AI 응답 생성 시 **프롬프트에 직접 반영**되어 일관된 브랜드 응답을 제공한다.

---

### 🎨 말투 유형

- **정중한 말투**
  - 공손하고 공식적인 응답 스타일
  - 고객 불만, CS 응대에 적합

- **친근한 말투**
  - 부드럽고 따뜻한 응답 스타일
  - 단골 고객 및 긍정 리뷰에 적합

- **캐주얼 / 브랜드 전용 톤**
  - 브랜드 성격을 반영한 자유로운 말투
  - 젊은 타겟 또는 마케팅 목적에 적합

---

### 🧠 AI 응답 생성 반영 구조

```text
[사장님 말투 선택]
        ↓
[프롬프트 톤 파라미터 적용]
        ↓
[KoBERT + LLM 분석]
        ↓
[RAG 기반 답글 생성]
```

## 2️⃣ 리뷰 상태 관리 (Review Status Management)

리뷰는 상태에 따라 구분되며,  
**미완료 상태의 리뷰에 대해서만 AI 답글 추천 기능을 제공**한다.  
이를 통해 중복 응답을 방지하고 리뷰 관리 효율을 높인다.

---

### 📌 리뷰 상태 구분

- **리뷰 작성 중**
  - 아직 답글이 등록되지 않은 상태
  - AI 답글 추천 기능 활성화

- **리뷰 작성 완료**
  - 답글이 이미 등록된 상태
  - AI 답글 추천 비활성화
  - 관리 완료 상태로 분류

---

### 🧠 처리 로직 개요

```text
리뷰 수신
   ↓
리뷰 상태 확인
   ↓
[리뷰 작성 중]  → AI 답글 추천 제공
[리뷰 작성 완료] → 관리 완료 처리
```

## 3️⃣ 리뷰 분석 & 응답 생성 (Review Analysis & Response Generation)

고객 리뷰는 **KoBERT + LLM**을 활용하여 분석되며,  
리뷰의 감정과 의도를 파악한 후 **RAG 기반 답글을 자동 생성**한다.

---

### 🧠 분석 항목

- **감정 분석**
  - 긍정 (Positive)
  - 부정 (Negative)
  - 중립 (Neutral)

- **리뷰 유형 분류**
  - 요청 (Request)
  - 불만 (Complaint)
  - 칭찬 (Praise)

---

### 🔄 처리 흐름

```text
리뷰 텍스트 입력
   ↓
KoBERT 감정 분석
   ↓
LLM 리뷰 맥락 / 의도 분석
   ↓
감정 + 유형 결과 통합
   ↓
RAG 기반 답글 생성
```

## 4️⃣ 답글 추천 및 선택 (Reply Recommendation & Selection)

AI는 리뷰 분석 결과를 기반으로 **여러 개의 답글 후보를 생성**하며,  
운영자는 상황에 맞는 답글을 **선택·수정하거나 직접 작성**할 수 있다.

---

### 📌 답글 후보 제공 방식

- 리뷰 대댓글에 대해 **복수 개의 AI 답글 후보 제공**
- 감정, 리뷰 유형, 말투 설정을 반영한 응답 생성
- 운영자가 가장 적절한 답글을 선택 가능

---

### 👤 운영자 선택 방식

- **추천 답글 그대로 사용**
  - AI가 생성한 답글을 수정 없이 즉시 등록

- **추천 답글 수정 후 등록**
  - AI 답글을 기반으로 일부 문구 편집 가능

- **직접 작성**
  - AI를 사용하지 않고 운영자가 직접 답글 작성

---

### 🔄 처리 흐름

```text
AI 답글 후보 생성 (N개)
   ↓
운영자 선택
   ├─ 추천 답글 그대로 등록
   ├─ 추천 답글 수정 후 등록
   └─ 직접 작성 후 등록
```

## 🚀 추가 기능 (Advanced Features)

본 서비스는 기본적인 리뷰 답글 자동화 기능 외에도,  
보다 정교하고 개인화된 응답 생성을 위한 **확장 기능**을 제공한다.

---

### 🔹 닉네임 분석 (Nickname Analysis)

- 고객 닉네임을 리뷰 텍스트와 함께 분석
- 닉네임 패턴을 기반으로 단골 여부 및 친밀도 추정
- 분석 결과에 따라 **답글 말투 자동 조정**


### 🔹 키워드 & 감정 분석 대시보드 (Keyword & Sentiment Dashboard)

- 리뷰 핵심 키워드 자동 추출
- 긍정 키워드
- 부정 키워드
- 사장님이 한눈에 확인 가능한 요약 제공

### 🔹 Few-shot 기반 답글 학습 (Few-shot Reply Adaptation)

- 사장님이 작성한 답글을 퓨샷 데이터로 누적
- 점점 사장님 스타일에 가까운 답글 생성

### 🔹 리뷰 이미지 분석

- 리뷰에 포함된 사진 분석
- 음식 상태
- 포장 상태
- 문제 상황 추정
- 이미지 분석 결과를 답글 생성에 반영


## 👤 사용자 흐름 (User Flow)

운영자는 아래 흐름에 따라 리뷰를 확인하고, AI가 추천한 답글을 선택/수정하여 등록한다.

 **1. 사장님 로그인**  
 **2. 리뷰 목록 확인**  
 **3. 미작성 리뷰 선택**  
 **4. AI 분석 결과 및 추천 답글 확인**  
 **5. 답글 선택 또는 수정**  
 **6. 답글 등록 → 리뷰 완료 처리**  

## 📈 기대 효과

본 서비스 도입을 통해 다음과 같은 효과를 기대할 수 있다.

- **반복적인 리뷰 응대 업무 자동화**  
  → 운영자의 수작업 부담 감소 및 운영 효율 향상

- **고객 응대 품질 향상**  
  → 리뷰 감정과 유형에 맞는 일관되고 자연스러운 답글 제공

- **브랜드 톤 일관성 확보**  
  → 말투 설정 및 Few-shot 학습을 통한 브랜드 고유 응답 스타일 유지

- **고객 만족도 및 재방문율 증가**  
  → 빠르고 정성 있는 응대로 고객 신뢰도 및 재이용 가능성 향상


## 🔮 향후 확장 계획

본 서비스는 초기 기능 구현 이후, 다음과 같은 방향으로 확장할 수 있다.

- **다국어 리뷰 자동 답글 지원**  
  → 글로벌 고객을 대상으로 한 리뷰 자동 응대 및 언어별 말투 최적화

- **플랫폼별 응답 톤 분리**  
  → 배달앱, 쇼핑몰 등 플랫폼 특성에 맞는 응답 말투 및 전략 적용

- **자동 응답 신뢰도 점수 제공**  
  → AI가 생성한 답글에 대해 신뢰도/적합도 점수를 함께 제공하여 운영자의 판단을 지원

- **답글 성과 분석 기능**  
  → 답글 등록 이후의 평점 변화, 재리뷰 여부 등을 분석하여 응답 효과 측정
